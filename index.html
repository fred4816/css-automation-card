User Onboarding

Automation to onboard new users.

  • 1. Enter the user's full name and pick an option from the suggested user names
  • 2. Pick an O365 licence from the drop-down box
  • 3. Enter a password or select the tick box to generate a random password
  • 4. Select the groups the user should be a member of
  • 5. Click the tick box for a VPN account if one is needed
  • 6. Click Submit
